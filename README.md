# Desafio – Verificação de Cartões
Aplicação web desenvolvida em ASP.NET Core com Razor Pages para identificação da bandeira e validação de números de cartões de crédito utilizando o algoritmo de Luhn.
Projeto criado como parte de um desafio prático, com foco em boas práticas, organização de código e uso de serviços.
## Funcionalidades
- Validação de cartão pelo algoritmo de Luhn
- Identificação da bandeira do cartão:
  - Visa
  - MasterCard
  - American Express
  - Discover
  - Diners Club
  - EnRoute
  - JCB
  - Voyager
  - Aura
  - Hipercard
- Aceita números com espaços e hífens
- Interface web simples e funcional
## Tecnologias Utilizadas

- .NET 7 / .NET 8
- ASP.NET Core
- Razor Pages
- C#
- HTML5
- Regex
- Git & GitHub
